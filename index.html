Chromebook Tester

Welcome to chromebooktester.com.
This site was designed to assist in the triage process of Chromebooks to enable faster repairs.
Despite its name and original intent, the features available here should work on any Operating System.

System Information:

This section shows which web browser you're using.
On ChromeOS, you can see which version of ChromeOS you're running by looking at the version of Chrome.

Microphone Test

The recorded audio will automatically be played back once recording is finished.
If you haven't previously allowed Chromebook Tester to use your microphone, you will be asked to do so.
Make sure that your microphone is turned on, and that your volume is turned up to an audible level before beginning the test.
You can adjust the recording length between 3 seconds to 300 seconds (5 minutes).

Keyboard Test

If you press any keyboard keys while on this page, they should be highlighted in green below. Be mindful that the actual key pressed will be highlighted regardless of your layout. That is to say, although the US-English QWERTY layout is shown below, if you were to press the "A" key on an AZERTY keyboard layout, the "Q" would be highlighted below. However, if you click on the text field and begin typing, you should see input based on your current keyboard layout.
